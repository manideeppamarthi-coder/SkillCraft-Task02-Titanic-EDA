# SkillCraft Data Science Internship – Task 02

## Titanic Dataset Exploratory Data Analysis

### Objective

Perform Exploratory Data Analysis (EDA) on the Titanic dataset to discover trends, relationships, and factors affecting passenger survival.

### Dataset

Titanic Training Dataset (`train.csv`)

### Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

### Project Workflow

1. Loaded and inspected the dataset.
2. Checked missing values and data types.
3. Generated descriptive statistics.
4. Explored relationships between variables.
5. Visualized survival patterns using charts and graphs.

### Key Insights

* Survival distribution among passengers.
* Impact of gender, age, and passenger class.
* Identification of important features related to survival.
